v7 — Resource Always at Top, Three Cases

Universal shape: resource header → highlight content. The module wrapper appears only when 2+ highlights cluster on one resource. The "Highlighted by …" strip appears only when more than one person highlighted — same person highlighting twice skips the strip entirely (the avatar repeats on every card already).
Rules
  • Always: resource header at the top of the item.
  • 1 highlight → flat in feed, no module. Resource header → divider → byline → pull-quote.
  • 2+ highlights, 1 highlighter → module wrapper, no highlighters strip (redundant). Resource → reel.
  • 2+ highlights, 2+ highlighters → module wrapper + highlighters strip. Resource → strip → reel.
