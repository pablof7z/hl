Behavior summary
  • 1 highlight on a resource → flat in feed (existing pull-quote card). No module.
  • 2+ highlights on the same resource → module appears, regardless of whether they're from the same person or many people.
  • Resource header adapts to artifact type: NIP-23 article (cover + author + title), web URL (favicon-y mark + domain + page title), podcast (cover + show + episode + play CTA), and so on. Same module shape, different fillings.
  • Kind pill ("Article" / "Web" / "Podcast") next to the author/source line makes the artifact type unmissable at a glance.
  • Highlighter avatar overflow: show first 3 + "+N", with the label collapsing to "Highlighted by Gigi, Anna and N others".
