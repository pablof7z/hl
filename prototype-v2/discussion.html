How do you apply "System 2" thinking in daily life?

After reading Part I, I'm curious how others are consciously engaging System 2. Kahneman makes it clear that we default to System 1 most of the time. What strategies have you found for deliberately engaging the slower, more analytical mode?

I've been using a simple rule: whenever I feel certain about something, I pause and ask "what would change my mind?" It's basically forcing System 2 to audit System 1's snap judgment. Surprisingly effective for everything from code reviews to hiring decisions.

Circling back after trying Bob's method for a few days. It works, but I noticed something — the question "what would change my mind?" only helps when I can actually generate alternatives. When I'm deeply embedded in a perspective, I can't even imagine what the contrary evidence looks like. That's when talking to someone from a different community is gold.
